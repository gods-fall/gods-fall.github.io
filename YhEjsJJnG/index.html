java学习笔记4

2022-03-12

逻辑控制
3种
顺序结构
分支结构
循环结构
if分支结构
if
针对关系表达式进行判断处理的分支操作
三类
if判断 if (布尔表达式) {条件满足时执行；}
if else判断 {条件不满足时执行；}
多条件 else if
switch
开关语句 对内容进行判断 数据（int char 枚举 String） 不能使用逻辑判断
switch（数值）{
case 数值 ：{
数值满足时执行；
[break]
}
case 数值：{
数值满足时执行；
[default:所有数值都不满足时执行]
}
}
如果没有追加break，程序会继续执行
while循环结构
重复执行操作
while （布尔表达式）{
条件满足时执行；
修改循环条件；
}
先判断后执行。优选
do{
条件满足时执行；
修改循环条件；
} while （布尔表达式）；
先执行后判断
for循环
for (定义循环的初始化数值；循环判断；修改循环数据){
循环语句的执行；
}
对于while和for 优先选择
明确循环次数 选for
不知道次数知道循环结束条件使用while
#循环控制
两个控制语句 break continue
break退出整个循环结构
continue 只是结束一次调用，后续判断不执行，直接进行后续
#循环嵌套
在一个循环语句嵌套其他的循环
循环嵌套的层次越多时间越多
